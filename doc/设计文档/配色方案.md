# 系统配色方案设计文档

## 1. 设计理念
本配色方案旨在打造一个专业、稳重且易于阅读的后台管理系统界面。通过控制颜色的饱和度和使用面积，减少视觉疲劳，提升用户体验。

- **简约性**：核心颜色不超过5种。
- **一致性**：全系统共用一套颜色变量。
- **专业感**：采用低饱和度的深蓝色作为主色调。

## 2. 核心配色

| 类别 | 颜色变量 | 色值 | 用途说明 |
| :--- | :--- | :--- | :--- |
| **主色** | `$primary-color` | `#2F54EB` | 品牌色，用于按钮、激活状态、重要链接 |
| **成功色** | `$success-color` | `#52C41A` | 用于成功提示、完成状态 |
| **警告色** | `$warning-color` | `#FAAD14` | 用于警告提示、待办状态 |
| **错误色** | `$error-color` | `#F5222D` | 用于错误提示、危险操作 |

## 3. 中性色（文字与背景）

| 类别 | 颜色变量 | 色值 | 用途说明 |
| :--- | :--- | :--- | :--- |
| **标题色** | `$title-color` | `#262626` | 用于主要标题、正文强调 |
| **正文色** | `$text-color` | `#595959` | 用于一般正文描述 |
| **次要色** | `$sub-text-color` | `#8C8C8C` | 用于辅助文字、提示信息 |
| **边框色** | `$border-color` | `#D9D9D9` | 用于分割线、边框 |
| **背景色** | `$bg-color` | `#F5F5F5` | 页面主体背景 |

## 4. 侧边栏（Sidebar）配色

侧边栏采用深色调以区分导航区与内容区。

| 类别 | 颜色变量 | 色值 | 用途说明 |
| :--- | :--- | :--- | :--- |
| **侧边栏背景** | `$menuBg` | `#001529` | 导航菜单背景色 |
| **菜单文字** | `$menuText` | `rgba(255, 255, 255, 0.65)` | 默认菜单文字颜色 |
| **激活文字** | `$menuActiveText` | `#FFFFFF` | 选中状态菜单文字颜色 |
| **悬停背景** | `$menuHover` | `#1890FF` | 鼠标悬停时的背景色 |

## 5. 实施方案
1. 所有的颜色值必须定义在 `xuanji-frontend/src/styles/variables.scss` 中。
2. 组件开发时严禁使用硬编码的色值，必须引用上述变量。
3. 保持 CSS 变量与设计文档同步。
